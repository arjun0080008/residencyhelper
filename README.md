# Residency Helper

This project lists information about internal medicine residency programs. The `residency_homepage.html` page loads `programs.json` and allows filtering by state, visa type and IMG percentage.

## Running
1. Start a simple HTTP server in the repository folder:
   ```bash
   python3 -m http.server 8000
   ```
2. Open your browser to `http://localhost:8000/residency_homepage.html`.

Filtering and details should now work without any additional setup.
